# VS Code Clone

A visual recreation of the VS Code interface built with HTML and CSS.

## Overview
- Static clone of VS Code's UI layout
- Built with HTML5 and CSS3
- Includes sidebar, editor area, and status bar design

## Quick Start
1. Clone the repo:
   ```bash
   git clone https://github.com/Ptejra1/VS-code-Clone.git
   ```
2. Open `VS clone/index.html` in your browser

## Structure
```
VS clone/
├── index.html
└── style.css
```
